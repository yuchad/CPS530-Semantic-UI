Installing Semantics-UI via Cloud9 (Ubuntu)

Prerequisite: Must have nodejs

1) Straight from the command line, run:

curl --silent --location https://deb.nodesource.com/setup_5.x | sudo bash - sudo apt-get install --yes nodejs

2) Run Gulp from command line

npm install -g gulp

3) Make a directory. Call it like, cps530RocksMySocks

4) Change to that directory and run:

npm install semantic-ui --save cd semantic/ gulp build
