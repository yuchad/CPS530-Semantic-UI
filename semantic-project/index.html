What is Semantic UI?

Semantic is a development framework that helps create beautiful, responsive layouts using human-friendly HTML. It is a UI component framework based around useful principles from natural language. It offers a very well organized documentation, plus a separate website that offers guides for getting started, customizing and creating themes.

How Popular is this Framework?

It became popular over the years. It is a framework that is easy to understand. Semantic UI is an ongoing effort to make building website much more semantic. In the near future it will be one of the most famous front-end framework out there.

Strength

Semantic class names make for a low barrier of entry, so even beginners can hit the ground running

Small file sizes and minimal load times because you can load only the components that you need; each has its own JS file and stylesheet

Versatile elements make for easy customization

Weakness

Very large packages when compared to Foundation and Bootstrap

Those with more complex design and development needs may find this framework lacking
